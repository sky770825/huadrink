# 部分手機顯示空白畫面 - 排查說明

若使用者回報開啟 https://huadrink.pages.dev/ 或後台頁面時出現**全白／空白畫面**，可請對方依以下方式嘗試：

## 使用者可自行嘗試

1. **改用系統瀏覽器開啟**
   - 若在 **Line**、**微信**、**Facebook** 等 App 內點連結，請改為**複製連結 → 貼到 Chrome 或 Safari** 開啟
   - App 內嵌瀏覽器對現代網頁支援較差，易出現空白

2. **改用 Chrome 開啟（Vivo / OPPO 等建議）**
   - Vivo 等中國品牌手機內建瀏覽器可能不相容，請至應用商店安裝 **Chrome** 後，用 Chrome 開啟連結

3. **清除快取並重試**
   - 設定 → 清除瀏覽記錄／快取 → 重新開啟連結

4. **關閉省流量／無痕模式**
   - 省流量或無痕模式可能阻擋部分資源載入

5. **更換網路**
   - 改用 Wi‑Fi 或不同電信網路測試

## 可能原因說明

| 原因 | 說明 |
|------|------|
| **App 內嵌瀏覽器** | Line、微信、QQ 等內建瀏覽器對 JavaScript 支援有限 |
| **Vivo / OPPO / 中國品牌手機** | 內建瀏覽器或 Funtouch/Origin OS 的 WebView 對部分網站支援較差，建議改用 **Chrome** 開啟 |
| **過舊手機／瀏覽器** | Android 8 以前、iOS 12 以前的裝置較易出現相容性問題 |
| **中國大陸網路** | `pages.dev` 為 Cloudflare 服務，在中國大陸部分網路（尤其中國移動）可能無法穩定連線 |
| **JavaScript 錯誤** | 特定裝置上執行時發生錯誤，導致畫面無法渲染 |

## 專案已做的改進

- **build target** 調整為支援較舊瀏覽器（Safari 12、Chrome 64+）
- **Error Boundary**：當發生錯誤時顯示排查說明，而非全白
- **noscript** 提示：停用 JavaScript 時顯示說明

## 若使用「電腦 Chrome」仍顯示「頁面載入發生問題」

代表發生 JavaScript 錯誤，可能原因：
- **網路限制**：公司、學校網路或 VPN 封鎖 Supabase / Cloudflare 連線
- **中國大陸**：部分網路無法連線至國外服務
- **瀏覽器擴充功能**：廣告攔截、隱私擴充可能阻擋 API 請求
- **無痕模式**：部分權限或儲存可能受限

**請對方：**
1. 點擊畫面上的「複製錯誤資訊」按鈕，將內容貼給您
2. 嘗試：關閉擴充功能、改用無痕/一般模式、換網路（如手機熱點）
3. 若在中國大陸，可嘗試 VPN 或換網路環境

## 若問題持續

可請對方提供：
- 手機型號與作業系統版本（或電腦與瀏覽器版本）
- 使用的瀏覽器（含版本）
- 連結是從哪個 App 開啟（Line、微信、直接輸入網址等）
- 若有「複製錯誤資訊」：錯誤詳情內容
