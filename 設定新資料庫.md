# huadrink 設定新資料庫

## 步驟一：取得 API 金鑰

1. 開啟 https://supabase.com/dashboard/project/sqgrnowrcvspxhuudrqc/settings/api
2. 複製 **Project URL**、**anon public**、**service_role** key
3. 編輯專案根目錄的 `.env`，填入：

```
VITE_SUPABASE_PUBLISHABLE_KEY=貼上 anon public key
SUPABASE_SERVICE_ROLE_KEY=貼上 service_role key
```

（VITE_SUPABASE_URL 和 PROJECT_ID 已設定好）

## 步驟二：建立資料表

1. 開啟 https://supabase.com/dashboard/project/sqgrnowrcvspxhuudrqc/sql/new
2. 複製 `supabase/huadrink-setup.sql` 的內容
3. 貼上後按 **Run** 執行
4. 若 API 無法存取 huadrink，請到 Project Settings > API > Exposed schemas 新增 `huadrink`

## 步驟三：建立管理員並匯入報名名單

在終端機執行（請替換成你的信箱和密碼）：

```bash
ADMIN_EMAIL=你的信箱 ADMIN_PASSWORD=你的密碼 npm run setup-new-db
```

或互動式輸入：

```bash
npm run setup-new-db
```

完成後即可用設定的帳密登入後台。
